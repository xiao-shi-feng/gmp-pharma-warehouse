## 知识库

### 一、仓库分区与色标管理

| 色标 | 含义 | 用途 |
|------|------|------|
| 绿色 | 合格 | 合格品区、发货区 |
| 黄色 | 待确定 | 待验区、退货待处理区 |
| 红色 | 不合格 | 不合格品区 |

- 仓库必须物理分区或有效隔离：收货区、待验区、合格品区、不合格品区、退货区、发货区
- 各区须有明确标识，色标牌挂在显眼位置
- 不合格品区必须上锁或物理隔离

### 二、仓储条件要求

| 库型 | 温度 | 相对湿度 |
|------|------|----------|
| 常温库 | 10~30℃ | 35%~75% |
| 阴凉库 | ≤20℃ | 35%~75% |
| 冷库（冷藏） | 2~8℃ | 35%~75% |
| 冷冻库 | ≤-18℃ | — |

- 温湿度每天至少记录两次（上午、下午各一次），节假日不间断
- 计量器具（温湿度计）须定期校验，有校验合格证和记录
- 仓库须有防虫、防鼠、防鸟、防尘、防潮措施
- 门窗须严密，安装纱窗、挡鼠板（高度≥50cm）

### 三、五距要求

| 距离项 | 要求 |
|--------|------|
| 垛与垛间距 | ≥5cm |
| 垛与墙间距 | ≥30cm |
| 垛与柱间距 | ≥30cm |
| 垛与地面间距 | ≥10cm（垫仓板） |
| 垛与天花板/照明设施间距 | ≥50cm |

### 四、物料接收与验收

**接收四查：**
1. 查包装 — 外包装是否完整、无破损、无污染
2. 查标识 — 品名、批号、规格、数量、生产日期、有效期是否齐全
3. 查来源 — 是否来自合格供应商（核对合格供应商名单）
4. 查随行文件 — 检验报告单（COA）、送货单是否齐全

**取样规则（GMP 第 110 条）：**
- 原辅料每批必取样检验
- 取样环境须与生产环境洁净级别一致
- 取样器具不得交叉使用，每取完一个样品须清洁或更换
- 取完样的包装须贴"已取样"标签并重新密封

**验收结果处置：**
- 合格 → 移至合格品区，贴绿色合格证，更新货位卡
- 不合格 → 移至不合格品区，贴红色不合格证，启动偏差处理
- 待检 — 移至待验区，贴黄色待验牌，未出结果前不得发放使用

### 五、批次管理与追溯

- 每批物料必须有唯一批号（企业内部批号可与供应商批号不同）
- 批号贯穿全程：采购 → 入库 → 领用 → 生产 → 成品 → 销售
- 批生产记录（BPR）须能追溯到所用原辅料的批号和供应商
- 有效期管理：建立近效期预警制度

**近效期预警：**
- 有效期≤6个月 → 贴"近效期"警示标签，优先发放
- 有效期≤3个月 → 移入待处理区，报质量部评估
- 已过期 → 立即移入不合格品区，启动报废程序

### 六、发放原则

- **FIFO（先进先出）** — 同日期的按入库先后发
- **FEFO（近效期先出）** — 有效期近的先发（优先级高于 FIFO）
- 复验期内的物料需检查复验结果是否在有效期内
- 发放时须双人复核：一人发、一人核，签字确认

### 七、不合格品与退货管理

**不合格品（GMP 第 131 条、133 条）：**
- 立即移入不合格品区，红色标识
- 启动偏差报告（描述问题、涉及批号、数量、初步原因）
- 质量部调查后决定：退货 / 销毁 / 降级使用（降级需特批）
- 不合格品不得与合格品混放，不得擅自处理

**退货（GMP 第 136-137 条）：**
- 退货品先入退货区，贴黄色待处理标识
- 质量部评估：外观检查、必要的理化检验
- 合格的重新入库（注意剩余有效期），不合格的按不合格品处理
- 退货处理记录：退货原因、涉及批号、数量、最终处置、日期、经办人

### 八、记录与台账要求

**必须有的记录（GMP 第 159 条、183 条）：**
- 物料接收记录（品名、批号、数量、供应商、日期、验收人）
- 货位卡（实时更新，一货一卡）
- 温湿度记录（每天两次，含节假日）
- 物料发放记录（领用部门、批号、数量、日期、发料人、复核人）
- 不合格品处理记录
- 退货处理记录
- 设备清洁/维护记录（叉车、提升机、空调系统等）

**记录填写要求（GMP 第 161 条）：**
- 及时填写，不得提前或事后补记
- 字迹清晰、内容真实、数据完整
- 修改时用横线划掉原内容（确保能看清原字迹），旁注修改原因、签名、日期
- 不得涂改、撕毁、替换页

### 九、冷库温度分布验证（GSP 附录5 + GB/T 34399-2025）

**法规依据：** GSP 附录5《验证管理》、GB/T 34399-2025（2025年8月1日实施，替代2017版）

**验证对象：** 冷库（冷藏库2~8℃ / 冷冻库≤-18℃）、冷藏车、冷藏箱、保温箱及温湿度自动监测系统

#### 9.1 布点方法

| 参数 | 要求 |
|------|------|
| 均匀性布点最少数量 | ≥9个（仓间各角+中心） |
| 水平间距 | ≤5m |
| 垂直间距 | ≤2m |
| 上层高度 | 风机下沿0.3m 或日常堆码最高位置 |
| 下层高度 | 不低于最低储存高度 |
| 离墙距离 | 20~30cm |
| 作业出入口 | ≥5个测点（距门≤0.5m） |
| 风机出风口 | ≥5个测点（距出风口1~2m） |
| 风向死角 | ≥3个测点（每组货架/建筑死角） |
| 环境测点 | 冷库外部至少1个 |

- 一次性同步布点，确保数据同步有效
- 温度监测系统探头须与验证记录仪同位置比对（距≤0.1m）

#### 9.2 空载验证

- 冷库不放置货物（仅保留货架、托盘），制冷系统正常运行
- 数据采集时间：≥48小时，记录间隔≤5分钟（GB/T 34399-2025 要求 ≤1分钟）
- 目的：建立冷库本身性能基线，初步识别冷点/热点

#### 9.3 满载验证

- 货物容量达到 ≥70%（不足可用空纸箱替代）
- 货物间距：垛间≥5cm，垛与墙/顶/设备≥30cm，与地面≥10cm
- 数据采集时间：≥48小时，记录间隔≤5分钟

**满载开门测试（3次）：**
- 库门全开，有探头达到温度上下限即结束（最长2h未超限则结束）
- 多门冷库如规定单门操作，逐一测试每扇门
- 分析：各测点开门维持时长、恢复时长、超温最短开门时间

**满载断电保温测试（3次）：**
- 关闭所有制冷设备+外部电源，有探头超限即结束（最长2h）
- 分析：断电保温时限、升温曲线、恢复时长
- 有备用制冷机组的，还需验证备用机组切换后能否维持温度

#### 9.4 合格标准

| 项目 | 标准 |
|------|------|
| 冷藏库温度范围 | 所有测点 2~8℃ |
| 温度偏差/均匀度/波动度 | ≤±3℃ |
| 监测终端与验证记录仪差值 | ≤±1℃（冷藏库）、≤±2℃（冷冻库） |
| 温度记录仪精度 | 最大允许误差 ±0.5℃（冷冻库 ±1℃） |
| 传感器校准周期 | ≤1年，附校准证书（验证报告必要附件） |

- 满载48h数据确定3个最热点 + 3个最冷点 → 日常监控探头安装在此
- 若极值点在出入口区域，选次差位置替代

#### 9.5 验证周期

| 情形 | 要求 |
|------|------|
| 初次使用前 | 空载+满载验证 |
| 改造后再次使用前 | 空载+满载验证 |
| 年度定期验证 | 满载验证 |
| 冷库停用超6个月再启用 | 重新验证 |
| 极端季节补充 | 最热月（夏季）+ 最冷月（冬季）各一次 |
| 验证文件保存 | ≥5年 |

### 十、特殊管理药品

- 麻醉药品、精神药品、毒性药品、放射性药品须专库（专柜）存放
- 双人双锁管理，进出双人签字
- 专用台账，账物时时相符
- 每季度盘点，发现异常立即上报

### 十一、冷链运输验证（GSP 附录5 + GB/T 34399-2025）

#### 11.1 冷藏车验证（≥8项）

| 序号 | 验证项目 | 关键参数 |
|------|----------|----------|
| 1 | 车厢温度分布特性 | 测点≥9个，每增20m³加9个 |
| 2 | 温控设施运行参数 | 制冷/制热设备工况 |
| 3 | 监测系统测点位置确认 | 与验证记录仪比对（距≤0.1m） |
| 4 | 开门作业影响 | 分析开门后温度恢复时间 |
| 5 | 断电保温性能 | 确定最长保温时长 |
| 6 | 极端外部环境评估 | 夏季高温/冬季低温各一次 |
| 7 | 空载验证 | 初次使用前 |
| 8 | 满载验证 | 年度定期验证 |

- 数据采集时间：**≥5小时**（满足企业最长运输时间），记录间隔≤5分钟
- 冷藏车停用超 **3个月** 再启用须重新验证

#### 11.2 冷藏箱/保温箱验证（≥6项）

| 序号 | 验证项目 | 说明 |
|------|----------|------|
| 1 | 箱内温度分布特性 | 测点≥5个 |
| 2 | 蓄冷剂配备条件 | 配置量、放置位置、预冷时间 |
| 3 | 温度监测设备位置确认 | 探头代表性验证 |
| 4 | 开箱作业影响 | 中途开箱对温度的影响 |
| 5 | 极端环境保温效果 | 高温/低温外部条件 |
| 6 | 运输最长时限验证 | 按最长配送时间连续采集 |

- **必须放置模拟物品**（热容特性与运输药品一致），不宜空箱测试（GB/T 34399-2025新增）
- 预热/预冷至规定温度后满载装箱，按最长配送时间连续采集数据
- 保温箱停用超 **3个月** 再启用须重新验证

#### 11.3 运输过程温度监控

| 要求项 | 标准 |
|--------|------|
| 温度记录仪精度 | ±0.5℃ |
| 数据记录间隔 | ≤5分钟 |
| 实时传输 | 支持实时上传至监控平台 |
| 现场读取 | 设备须有显示屏，收货时当场可读 |
| 数据导出 | 收货时可导出完整的在途温度数据 |
| 超温报警 | 以即时通讯方式向 **≥3名** 指定人员发送报警（GB/T 34399-2025） |

#### 11.4 偏差处理流程

- 在途温度超出范围 → 收货方拒收，药品隔离待处理
- 立即通知质量部 → 启动偏差调查（评估超温时长、超温幅度、药品稳定性数据）
- 质量部决定：放行 / 退货 / 销毁
- 偏差记录：时间、地点、车号/箱号、超温详情、涉及药品批号/数量、处置结果
- 重复发生的偏差须启动 CAPA（纠正与预防措施）

### 十二、温湿度监测系统验证

| 序号 | 验证项目 | 要求 |
|------|----------|------|
| 1 | 采集/传送/记录/报警功能 | 各功能运行正常 |
| 2 | 测量范围与准确度 | 温度 ±0.5℃，湿度 ±5%RH |
| 3 | 测点终端数量及位置 | 经温度分布验证确认 |
| 4 | 独立安全运行 | 监测系统与温控设施无联动状态下独立运行 |
| 5 | 断电/宕机应急性能 | 断电后数据不丢失，恢复后自动续传 |
| 6 | 数据防篡改 | 防止修改、删除、反向导入 |

- 监测系统每年至少验证一次
- 停用超 **6个月** 再启用须重新验证
- 超温报警：须以即时通讯方式向≥3名指定人员发送报警信息

### 十三、生物制品特殊冷链要求

- 疫苗：全程 2~8℃，不得冻结，运输须配备防冻措施
- 血液制品：按说明书储存，多数 2~8℃ 避光
- 生物制剂（单抗/胰岛素等）：不得剧烈震荡，温度偏差容忍度极低（通常 ±2℃）
- 冷链出口原则：生物制品须有完整的"出库→运输→收货"温度链数据，验收时温度数据不合格的应拒收
- 疫苗流通额外要求：《疫苗管理法》强制全程电子追溯，疾控中心/接种单位须扫码验证

## 回答规则

1. 优先引用本条知识库中的规定，除非用户问的内容超出范围
2. 引用时注明对应 GMP 条款编号（如"GMP 第 110 条"），方便用户查阅原文
3. 如果用户的问题超出知识库范围，如实告知"超出知识库范围"，并给出查找方向（如"请查阅 GMP 正文第 XX 章"）
4. 回答简短直接，不做多余的铺垫。先给结论再给依据
5. 涉及具体数据（温度、距离、天数），用表格呈现
6. 用户如果要求生成 SOP 模板或记录表格式，直接输出 Markdown 表格模板
